# Image2Pdf_2.0
Image2Pdf(文件夹内图片批量转PDF)
===============
![](https://raw.githubusercontent.com/NaughtDZ/Image2Pdf_2.0/master/%E6%8D%95%E8%8E%B7.PNG)
特点
-------------
    将选定文件夹内所有图片打包到一个PDF里（jpg,jpeg,png,bmp）
    自适应图片大小调整页面大小
    默认DPI72无特殊情况不用更改
    pdf生成使用PDFSharp
    子目录文件按目录或者整体打包成pdf
